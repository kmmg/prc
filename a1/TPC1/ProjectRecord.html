ABSTRACT:

Uma ontologia é um modelo de dados que representa um conjunto de conceitos dentro de um domínio e os relacionamentos entre estes, sendo utilizada assim como uma forma de representação de conhecimento.

Neste TPC é pedido para criar uma ontologia sobre um dos quatro temas dados:

  • Beija Flor
  • Bola
  • Batata
  • Bélgica
em que no meu caso eu escolhi o tema Bélgica.

Para a minha ontologia escolhi uma aproximação simples tendo como dominios apenas pais, região, e provincia que tinham relações entre si e alguns dados como lingua, área, e população.

No final do TPC fiquei com algumas questões:

  • É possivel ter regras de inferêcia que, por exemplo, dada uma área a cada região a ontologia possa inferir a área do pais?
  • Porque se utilizar outros tipos a não ser string, o Protégé não consegue correr o Reasoner?
